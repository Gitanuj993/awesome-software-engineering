# Install Git on Linux

## Debian / Ubuntu

Update package list:

```bash
sudo apt update
```

Install Git:

```bash
sudo apt install git
```

Verify installation:

```bash
git --version
```

---

## Fedora

Install Git:

```bash
sudo dnf install git
```

Verify installation:

```bash
git --version
```

---

## CentOS / RHEL

Install Git:

```bash
sudo dnf install git
```

If using an older version:

```bash
sudo yum install git
```

Verify installation:

```bash
git --version
```

---

## Arch Linux

Install Git:

```bash
sudo pacman -S git
```

Verify installation:

```bash
git --version
```

---

## openSUSE

Install Git:

```bash
sudo zypper install git
```

Verify installation:

```bash
git --version
```

---

## Alpine Linux

Install Git:

```bash
sudo apk add git
```

Verify installation:

```bash
git --version
```

---

## Verify Installation

Run:

```bash
git --version
```

Example output:

```text
git version 2.50.1
```
