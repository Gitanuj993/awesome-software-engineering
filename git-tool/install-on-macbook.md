# Install Git on macOS

## Method 1: Install with Homebrew (Recommended)

Update Homebrew:

```bash
brew update
```

Install Git:

```bash
brew install git
```

Verify installation:

```bash
git --version
```

---

## Method 2: Install with Xcode Command Line Tools

Install Xcode Command Line Tools:

```bash
xcode-select --install
```

Follow the installation prompts.

Verify installation:

```bash
git --version
```

---

## Method 3: Install from Official Installer

1. Visit https://git-scm.com/download/mac
2. Download the latest macOS installer.
3. Run the installer.
4. Complete the installation.

Verify installation:

```bash
git --version
```
