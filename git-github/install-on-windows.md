# 1. Install Git on Windows

## Download

Visit:

https://git-scm.com/

Download the latest Windows installer.

## Installation

During installation, keep the default options unless you know why you're changing them.

After installation, open:

- Git Bash
- PowerShell
- Windows Terminal
- CMD

---

## Verify Installation

```bash
git --version
```

Example:

```text
git version 2.50.1
```
